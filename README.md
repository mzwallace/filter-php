library
=======

A shared PHP Library that contains our domain / business logic

To include this package via composer in other projects
add the following to the composer.json file:

```
// composer.json
{
  "repositories": [
      {
          "type": "vcs",
          "url": "https://github.com/mzwallace/library"
      }
  ],
  "require": {
      "mzwallace/library": "*"
  }
}
```
